Материалы, ДЗ и практика Видео Ещё видео Селекторы
  1. Скачать Node.js с сайта. Версия 18. Усьтановить
  2. Перезапустить vs code
  3. В vs code Файл->Новое окно->Клонирование репозитория git->выбрать репозиторий с JS->отрыть
  4. В vs code открыть терминал (ctrl+~)
  5. В терминале написать 'npm i' и нажать enter
  6. В терминале написать 'npm run dev', нажать enter, перейти по ссылке
